What Is Invimtory?

Invimtory is a sales and inventory tracking system that uses an easily navigatible interface to allow users to easily add customers and orders to their accounts. It is also a great tool that can efficiently list complex and lengthy inventory kits to ensure shipment accuracy.

settings
How does it work?

Invimtory is powered by Javascript, HTML, and SASS. Using the AngularJS framework and a Firebase database, creating, saving, updating and deleting documents with an interactive and user friendly UI is fun and easy. Check out the github repo here for more detail and technologies.
